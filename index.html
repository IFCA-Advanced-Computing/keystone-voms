Keystone VOMS authentication module

What is this?

This module is a WSGI middleware that will provide VOMS authentication into OpenStack Keystone as a external authentication plugin. With this module it is possible to:

  • As a resource provider, enable users from a given VO to get access to a OpenStack installation.
  • As a user, use the VOMS credentials to authenticate against a VOMS aware OpenStack cloud using the standard tools, without any modifications (using the VOMS authentication plugin).
